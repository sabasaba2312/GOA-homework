(ტექსტურ ფაილში) თქვენი სიტყვებით ახსენით რა არის input და output, მოიყვანეთ მაგალითები და დაურთეთ სურათები

რა არის input?
input - არის შეტანა/შეყვანა
output - გამოტანა

input არის ის რაც ჩვენ კომპიტერში შეგვყავს ტექსტი, კოდი ან რაიმე სახის მონაცემი 
output - არის რაც გამოდის შედეგით სახით 

მაგ: შევქმენით ცვლადები და მივანიჭეთ მნიშვნელობები ეს არის input რაც ჩვენ შევიყვანეთ კომპიუტერში. ხოლო output მა ეს შედეგი გამოგვიტანა ტერმინალში, ბრაუზერში ან ასშ.
name = input(what is your name: ?)

print(name)

ჩვენ შევქმენით ცვლადი რაშიც გამოვიყენეთ input 
და კოდის გაშვების დროს ტერმინალმა გამოგვიტანა იგი (შეგვეკითხა)
