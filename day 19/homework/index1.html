GOA

GOA არის (მიზნებზე ორიენტირებული აკადემია) მსოფლიოში საუკეთსო აკადემია ყველაზე სანდო და ერთადერთი აკადემია, რომელიც პერსონალურად მოგიდგება და დაგეხმარება ნებისმიერი ცხოვრებისეული პრობლემის მოგვარებაში. დაგისახავს დიად მიზნებს და ბოლომდე მოგეხმარება მათ მიღწევაში მე ვსწავლობ გოაში პროგრამირებას.

რა მომწონს გოაში

მე გოაში მომწონს ყველაფერი რადგან მეხმარება მომავლის გალამაზებაში ჩვენს წარმატებების მისაღწევად ყველაფერს აკეთბენ მენტორები ლიდერები მინი ლიდერები და ბავშვებიც კი რადგან გოა არის მსოფლიოში საუკეთესო აკადემია. არის მგეობრული გარემო რაც ხელს მიწყობს რომ გავხდე ჩადი და ვიცხოვრო ბედნიერად. გოაში შესწავლა არის ძალიან მაღალ ხარისხსში რაც ყველა ბავშვს მოსწონს რადგან ულამაზებთ მომავალს ასეთი აკადემია.მე გოაში გავიჩინე ძალიან ბევრი მეგობარი რაც მე ძალიან მახარებს

HTML

ჰიპერტექსტის მარკირების ენა (HTML) არის სტანდარტული მარკირების ენა[a] დოკუმენტებისთვის, რომლებიც შექმნილია ვებ ბრაუზერში გამოსატანად. ის განსაზღვრავს ვებ კონტენტის შინაარსს და სტრუქტურას. მას ხშირად ეხმარება ისეთი ტექნოლოგიები, როგორიცაა Cascading Style Sheets (CSS) და სკრიპტირების ენები, როგორიცაა JavaScript, პროგრამირების ენა.ვებ ბრაუზერები იღებენ HTML დოკუმენტებს ვებ სერვერიდან ან ლოკალური მეხსიერებიდან და გადააქვთ დოკუმენტები მულტიმედია ვებ გვერდებზე. HTML აღწერს ვებ გვერდის სტრუქტურას სემანტიკურად და თავდაპირველად მოიცავდა მის გარეგნობას. HTML ელემენტები არის HTML გვერდების სამშენებლო ბლოკები. HTML კონსტრუქციებით, სურათები და სხვა ობიექტები, როგორიცაა ინტერაქტიული ფორმები, შეიძლება იყოს ჩასმული რენდერ გვერდზე. HTML საშუალებას გაძლევთ შექმნათ სტრუქტურირებული დოკუმენტები ტექსტის სტრუქტურული სემანტიკის მითითებით, როგორიცაა სათაურები, აბზაცები, სიები, ბმულები, ციტატები და სხვა ელემენტები. HTML ელემენტები გამოსახულია ტეგებით, რომლებიც იწერება კუთხის ფრჩხილების გამოყენებით. ტეგები, როგორიცაა img და input პირდაპირ ავრცელებენ შინაარსს გვერდზე. სხვა ტეგები, როგორიცაა p და /p, აკრავს და გვაწვდის ინფორმაციას დოკუმენტის ტექსტის შესახებ და შეიძლება შეიცავდეს ქვეელემენტის ტეგებს. ბრაუზერები არ აჩვენებენ HTML ტეგებს, მაგრამ იყენებენ მათ გვერდის შინაარსის ინტერპრეტაციისთვის.

CSS

Cascading Style Sheets (CSS) არის სტილის ფურცლის ენა, რომელიც გამოიყენება მარკირების ენაზე დაწერილი დოკუმენტის პრეზენტაციისა და სტილის დასაზუსტებლად, როგორიცაა HTML ან XML (მათ შორის XML დიალექტები, როგორიცაა SVG, MathML ან XHTML).[2] CSS არის მსოფლიო ქსელის ქვაკუთხედი ტექნოლოგია, HTML და JavaScript-თან ერთად.[3]CSS შექმნილია იმისათვის, რომ უზრუნველყოს შინაარსისა და პრეზენტაციის გამიჯვნა, განლაგების, ფერების და შრიფტების ჩათვლით.[4] ამ განცალკევებას შეუძლია გააუმჯობესოს კონტენტის ხელმისაწვდომობა, ვინაიდან კონტენტი შეიძლება დაიწეროს მის პრეზენტაციაზე ზრუნვის გარეშე; უზრუნველყოს მეტი მოქნილობა და კონტროლი პრეზენტაციის მახასიათებლების დაზუსტებაში; ჩართეთ მრავალ ვებ გვერდს გააზიაროთ ფორმატირება შესაბამისი CSS-ის ცალკე .css ფაილში მითითებით, რაც ამცირებს სტრუქტურულ შინაარსში სირთულეს და განმეორებას; და ჩართეთ .css ფაილის ქეშირება, რათა გაუმჯობესდეს გვერდის ჩატვირთვის სიჩქარე გვერდებს შორის, რომლებიც იზიარებენ ფაილს და მისი ფორმატირება.ფორმატირებისა და კონტენტის გამიჯვნა ასევე შესაძლებელს ხდის ერთი და იგივე მარკირების გვერდის სხვადასხვა სტილში წარმოდგენას სხვადასხვა რენდერის მეთოდისთვის, როგორიცაა ეკრანზე, ბეჭდვით, ხმით (მეტყველებაზე დაფუძნებული ბრაუზერის ან ეკრანის წამკითხველის საშუალებით) და ბრაილიზე დაფუძნებულ ტაქტილურ მოწყობილობებზე. CSS ასევე აქვს წესები ალტერნატიული ფორმატირებისთვის, თუ კონტენტზე წვდომა ხდება მობილური მოწყობილობიდან.[5]სახელწოდება კასკადური მომდინარეობს მითითებული პრიორიტეტის სქემიდან, რათა დადგინდეს, რომელი დეკლარაცია გამოიყენება, თუ ქონებიერთზე მეტი დეკლარაცია ემთხვევა კონკრეტულ ელემენტს. ეს კასკადური პრიორიტეტის სქემა პროგნოზირებადია.

JavaScript

JavaScript, ხშირად შემოკლებით JS, არის პროგრამირების ენა და მსოფლიო ქსელის ძირითადი ტექნოლოგია, HTML და CSS-თან ერთად. ვებსაიტების ოთხმოცდაცხრა პროცენტი იყენებს JavaScript-ს კლიენტის მხარეს ვებგვერდის ქცევისთვის.[10]ვებ ბრაუზერებს აქვთ გამოყოფილი JavaScript ძრავა, რომელიც ახორციელებს კლიენტის კოდს. ეს ძრავები ასევე გამოიყენება ზოგიერთ სერვერზე და სხვადასხვა აპში. ყველაზე პოპულარული გაშვების სისტემა არაბრაუზერის გამოყენებისთვის არის Node.js. JavaScript არის მაღალი დონის, ხშირად დროულად შედგენილი ენა, რომელიც შეესაბამება ECMAScript სტანდარტს.[11] მას აქვს დინამიური აკრეფა, პროტოტიპზე დაფუძნებული ობიექტზე ორიენტაცია და პირველი კლასის ფუნქციები. ეს არის მრავალ პარადიგმა, მხარს უჭერს მოვლენებზე ორიენტირებულ, ფუნქციურ და იმპერატიულ პროგრამირების სტილებს. მას აქვს აპლიკაციის პროგრამირების ინტერფეისები (API) ტექსტთან, თარიღებთან, რეგულარულ გამონათქვამებთან, მონაცემთა სტანდარტულ სტრუქტურებთან და დოკუმენტის ობიექტის მოდელთან (DOM) მუშაობისთვის.ECMAScript სტანდარტი არ შეიცავს რაიმე შეყვანას/გამომავალს (I/O), როგორიცაა ქსელი, საცავი ან გრაფიკული საშუალებები. პრაქტიკაში, ვებ ბრაუზერი ან სხვა გაშვების სისტემა უზრუნველყოფს JavaScript API-ებს I/O-სთვის. მიუხედავად იმისა, რომ Java და JavaScript მსგავსია სახელითა და სინტაქსით, ეს ორი ენა განსხვავებულია და მნიშვნელოვნად განსხვავდება დიზაინით.
