HTML

ჰიპერტექსტის მარკირების ენა არის მარკირების ენა დოკუმენტებისთვის რომელიც შეიქმნა გამოსატანად web browser-ში. ის განსაზღვრავს სტრუქტურას და კონტენტის შინაარს.მას ეხმარება ტექნოლოგიები-CSS და JavaScript. html შექმნა Tim Berners-Lee.

CSS

CSS არის „კასკადური სტილის ფურცლების“ აბრევიატურა.ეს არის კომპიუტერული ენა ვებ გვერდების (HTML ან XML) განლაგებისა და სტრუქტურირებისთვის და ის გამოიყენება ვებ გვერდზე სტილის დასამატებლად ბრაუზერში საიტის ჩვენების კარნახით.CSS შექმნა Håkon Wium Lie.

javascript

JavaScript არის ტექსტზე დაფუძნებული პროგრამირების ენა,გამოიყენება როგორც კლიენტის, ასევე სერვერის მხარეს,ის საშუალებას გვაძლევს გავხადოთთ ვებ გვერდები ინტერაქტიული.HTML და CSS არის ენები, რომლებიც სტრუქტურას და სტილს ანიჭებენ ვებ გვერდებს, JavaScript აძლევს ვებ გვერდებს ინტერაქტიულ ელემენტებს.და მას ხდის მშვენიერს.javascript შექმნა Brendan Eich.
